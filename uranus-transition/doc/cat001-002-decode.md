# CAT02报
## 原始报文
```txt
20181113-14:58:00 00 00 02 00 0B D4 08 22 01 30 FB A3 94 75 C8 
```

## 译码
### SOH 
- 0 = 0x00 = 0b00000000
- 1 = 0x00 = 0b00000000
    - V = 0 = SOH (ASCII)
 
### CAT
- 2 = 0x02 = 0b00000010  
    - DEC = 2

### LEN
- 3 = 0x00 = 0b00000000
- 4 = 0x0B = 0b00001011
    - DEC = 0000000000001011 = 11

### FSPEC
- 5 = 0xD4 = 0b11010100
    - FRN 1 2 4 6 
        - PRESENCE 
    - FX
        - NO EXTENSION


### F1 Data Source Identifier 
- 6 = 0x08 = 0b00001000
- 7 = 0x22 = 0b0010001
    - SAC = 00001000
    - SIC = 00100010

### F2 Message Type
- 8 = 0x01 = 0b00000001
    - DEC = 1 = North marker message
  
### F4 Time of Day
- 9 = 0x30 = 0b00110000‬
- 10 = 0xFB = 0b‭11111011‬
- 11 = 0xA3 = 0b‭10100011
    - DEC = 00110000‬11111011‬10100011 = ‭3210147‬ / 128 = 25079 s = UTC 6:57:59


### Station Configuration Status 
- 12 = 0x94 = 0b‬‭10010100
    - STATUS = 1001010
    - FX = 0 = NO EXTENSION

### THALES的报尾
- 13 = 0x75 = 0b01110101‬‭11001000‬
- 14 = 0xC8 = 0b‭11001000‬
